# CIRCERO Repository

This is the repository addon for CIRCERO Kodi addons. Installing this repository addon allows you to easily install and receive updates for all CIRCERO addons.

## Installation

1. Download the repository addon ZIP file
2. In Kodi, go to **Settings** → **Add-ons** → **Install from zip file**
3. Select the downloaded repository ZIP file
4. Wait for the installation confirmation

## Included Addons

### metadata.stash.python
Multi-scraper metadata addon supporting 7 different sources:
- Stash
- AEBN
- Brazzers
- FakeHub (FakeTaxi, FakeHostel, PublicAgent)
- CzechHunter (Debt Dandy, Dirty Scout)
- GayWire (Guy Selector)
- PrimalFetish Network
